4. Admission Enquiry Chat Bot Project แชทบอทที่ช่วยนักเรียนที่คาดหวังในกระบวนการรับเข้ามหาวิทยาลัยโดยการตอบคำถามและให้ข้อมูลเกี่ยวกับโปรแกรมเฉพาะ

Intro
การศึกษาในประเทศไทยมีความเหลือมลำในการศึกษา บ้างโรงเรียนมีครูแนะแนวการศึกษาที่ดี บ้างโรงเรียนก็มีแต่แนะนำแค่ในสมัยที่เขาเรียน บ้างโรงเรียนไม่มีการแนะนำเลย
แต่สมัยนี้มีเทคโนโลยีที่ช่วยให้การศึกษามีความเป็นมากขึ้น และการศึกษามีความเหลือมลำน้อยลง

Pain Point
การศึกษามีความเหลือมลำ
การศึกษาไม่มีการแนะนำ
การศึกษามีการเปลี่ยนแปลงบ่อยครั้ง
การศึกษามีการเปลี่ยนแปลงที่ไม่คาดคิด

Target
นักเรียนที่กำลังจะเข้ามหาวิทยาลัย

Features (Main)
แชทบอทที่ช่วยนักเรียนที่คาดหวังในกระบวนการรับเข้ามหาวิทยาลัยโดยการตอบคำถามและให้ข้อมูลเกี่ยวกับโปรแกรมเฉพาะ

Features (Additional)
นำคะแนนที่ได้มาคำนวณว่าจะเข้ามหาวิทยาลัยได้หรือไม่
แนะนำว่าควรไปศึกษาด้านไหนเพิ่มเติม
แนะนำว่าควรเรียนสายอะไร
แนะนำว่าควรเรียนสายอะไรเพื่อที่จะได้งานที่ดี
แนะนำว่าควรเรียนสายอะไรเพื่อที่จะได้งานที่มีโอกาสได้เรียนต่อต่างประเทศ
