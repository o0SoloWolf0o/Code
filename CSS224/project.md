# ขั้นตอนการแก้ปัญหา TSP ด้วยใช้ Java ประกอบด้วย:

1.การเขียนโปรแกรมสำหรับสุ่มเลือกลำดับเมืองและปรับปรุงลำดับโดยใช้วิธีการ simulated annealing.

2.การเขียนโปรแกรมสำหรับส่งลำดับของเมืองผ่านไปทุก node บน network ผ่าน TCP/IP โดยใช้ Java Sockets.

3.การเขียนโปรแกรมสำหรับบวกระยะไปพร้อมกันในขั้นตอนแรก และบวกระยะในท้ายที่สุดเมื่อครบสี่เมือง.

4.การติดตั้งและการใช้งาน Docker เพื่อใช้ในการแก้ปัญหา TSP.

5.การทดสอบโปรแกรมและวิเคราะห์ผลการทำงานเพื่อให้แน่ใจว่าโปรแกรมทำ
